# Team-Brown-CMS-Project\

## Link: https://jongryn.github.io/Team-Brown-CMS-Project/

Technologies: HTML/CSS, Bootstrap, jQuery, AJAX, Sequelize, MySQL, Node.js, Firebase, RESTful API, MongoDB, and Express

The Project Manager App allows users to use an interactive calendar to set events, communicate with other users through the chat box, users can mark off items on the to-do list, and insert contacts into the contact list.
